Analyser by usm-data-analysis

A library to work with files generated by CLASTool

This is a small set of classes to simplify the analysis of the ROOT files generated by the CLASTool package. The provided methods allow to extract the necessary information in a more natural way.

Dependencies

You need a proper ROOT installation, and also CLASTool. Read installation instructions for more details.

Install

The complete guides to install ROOT and CLASTool can be found in our Organization's wiki:

  • Installing ROOT
  • Installing CLASTool
Then, you need to get the Analyser source code, by downloading the packaged code or using the git command line (this last one is the preferred method), and follow the instruction in the wiki:
  • Installing Analyser
The test directory has examples of how to use the library in your applications.

Code Documentation

You can find the complete documentation for the Analyser classes in our Reference Guide.
